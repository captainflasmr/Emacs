#+title: Spare Keybindings — Reference

* Overview

Keybindings that are either unbound or bound to rarely-used default commands.  Good candidates for repurposing.

* Global Keys

** M-<letter> — rarely-used defaults

These are default Emacs bindings for commands most people seldom invoke:

| Key | Current binding                | Notes                                                                    |                      |
|-----+--------------------------------+--------------------------------------------------------------------------+----------------------|
| M-r | move-to-window-line-top-bottom | window scrolling; overridden in eshell/shell modes (Emacs-DIYer/init.el) |                      |
| M-t | transpose-words                | prose editing                                                            |                      |
| M-~ | not-modified                   | rarely used                                                              |                      |
| M-  |                                | shell-command-on-region                                                  | pipe region to shell |
