#+title: Spare Keybindings — Reference

* Overview

Keybindings that are either unbound or bound to rarely-used default
commands.  Good candidates for repurposing.

* Global Keys

** M-<letter> — rarely-used defaults

These are default Emacs bindings for commands most people seldom invoke:

| Key | Current binding               | Notes                        |
|-----|-------------------------------|------------------------------|
| M-c | capitalize-word               | word transformation          |
| M-r | move-to-window-line-top-bottom| window scrolling             |
| M-t | transpose-words               | prose editing                |
| M-~ | not-modified                  | rarely used                  |
| M-| | shell-command-on-region       | pipe region to shell         |

** C-c <letter> — free slots

None of these are currently bound:

  C-c b    C-c d    C-c e    C-c n    C-c r
  C-c t    C-c u    C-c v    C-c x

* Global keys explicitly unbound

| Key   | Notes                               |
|-------+-------------------------------------|
| C-t   | freed for package use (dired C-t *) |
| C-h h | explicitly unset                    |
