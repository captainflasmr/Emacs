This document contains the release notes for offline-packages

* Versions

** <2026-05-11>

Deduplicated and refined starter configs

- Deleted =starters/emacs-28.2.el= (identical to =emacs-28.1.el=) and
  =starters/emacs-30.1.el= (identical to =emacs-30.2.el=) — kept the latest
  version of each series.
- Removed =dape=, =treemacs=, =demap=, and =protobuf-mode= blocks from
  =starters/emacs-30.2.el= — these now live exclusively in =coding.el= to
  eliminate cross-file duplication. =emacs-30.2.el= header updated to point
  users to load both starters.
- Updated =emacs-28.1.el= header to document that it covers the 28.x series
  and note the =coding.el= companion.
- Updated =README.org= to reflect the current starter file set and cross-
  reference =coding.el= from the per-version starter description.

** <2026-05-07>

Centralised backups and starter naming

- Changed =setup.sh= to stash all backups for each install into a single =~/.emacs.d/.backups/<STAMP>= directory instead of littering =.pre-toolkit-*= suffixes across =~/.emacs.d/= and =~/=. Mirror extractions from =$HOME= go into a =home-mirrors/= subdirectory. Old installs can now be pruned by deleting a single directory.
- Changed =rollback.sh= to restore from the new =.backups/= directory layout. Still recognises legacy =.pre-toolkit-*= suffixes from older installs. Current post-rollback state is archived under =.backups/<NOW>-rolledback/= so a roll-forward is still possible.
- Renamed =coding-starter.el= → =init-starter-coding.el= so all starter files share the =init-starter-= prefix and group together in directory listings on the target.

** <2026-05-06>

Add demap, htmlize, and org-bootstrap-publish; enhance starter configs

- Added =demap= and =htmlize= to all package lists (=packages/emacs-*.el=)
- Added =org-bootstrap-publish= to =local-packages/=, replacing the =~/source/repos/= load-path dependency
- Updated =init.el= to load =org-bootstrap-publish= from =local-packages= via =user-emacs-directory= instead of hardcoded =~/source/repos/= path
- Enhanced =treemacs= config in starters with =:demand t=, full =setq= customisation, root face attribute, current-line overlay, and project-only toggle
- Added =demap= config with diff-hl integration and custom diff faces to =emacs-30.1.el= and =emacs-30.2.el= starters
- Added =windmove-repeat-map= with h/j/k/l repeat navigation to =emacs-30.1.el= and =emacs-30.2.el= starters
- Alphabetized all package lists and fixed duplicate =dumb-jump= entry in =emacs-28.1.el=
